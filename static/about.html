Speech synthesis with WaveNets

DISCLAIMER: To run the app in your browser, JavaScript's WebAudio API needs a secure origin. For this reason, the API seems to have quite an unstable behavior. In case you are experiencing problems, please run the app in your localhost.

This is a web application that let's users convert text into speech by using an OCR + WaveNets as the speech synthesis engine. This is done by showing the camera some image of text, which then gets converted into a string, and fed into the neural network. Due to the complexity of this task, there are some performance problems that I'm still working on. Some of these issues, along with future ideas for useful applications are elaborated on my github.

For more details or collaboration, please see my Github repo above.
